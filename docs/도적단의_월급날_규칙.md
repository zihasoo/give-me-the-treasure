# 도적단의 월급날 (Give Me The Treasure) — 규칙

> 코리아보드게임즈 한국어판 규칙서 v1.0.0 기준.

---

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 영문명 | Give Me The Treasure (원작 Susabi Games, 한국어판 코리아보드게임즈 2024) |
| 인원 | 2~4명, 항상 2팀 대결 |
| 플레이 시간 | 15~30분 |
| 장르 | 심리 / 블러핑 / 케이크 나누기(I-cut-you-choose) |
| 승리 조건 | 종료 단계에 30코인 이상 보유한 팀이 있으면, 그 시점에 더 많은 코인을 가진 팀이 승리 |
| 연습 룰 | 첫 플레이 시 20코인 승리 + 리더 효과 비활성화 추천 |

**핵심 루프**: 분할 팀이 보물 5장을 두 묶음으로 나누되 1장은 뒷면(비공개)으로 둠 → 선택 팀이 한 묶음을 고름 → 양 팀이 보물을 세트로 환금해 코인 획득 → 역할 교대. 상대의 "꾀"를 간파하는 것이 핵심.

## 2. 인원 구성 (팀)

| 전체 인원 | 팀 구성 |
|-----------|---------|
| 4인 | 2 vs 2 |
| 3인 | 1 vs 2 |
| 2인 | 1 vs 1 |

- 같은 팀끼리는 게임 내내 자유롭게 논의 가능.

## 3. 구성물

총 카드 44장(보물 35 + 도우미 9) + 토큰류.

| 구성물 | 수량 | 비고 |
|--------|------|------|
| 보물 카드 | 35장 | 일반 28 + 특수 7 (아래) |
| 도우미 카드 | 9장 | 캐릭터 9종, 게임당 팀별 2장씩만 사용 |
| 간부 타일 | 4개 | 플랭키 · 척 · 와이즈 · 죠 |
| 동전 | 24개 | 1코인(은) ×12 + 5코인(금) ×12 (합계 72코인) |
| 리더 토큰 | 2개 | 팀별 리더 표시 (리더 효과 활성화) |
| 팀 표시 막대기 | 1개 | 빨간 화살표가 분할 팀(주도권)을 가리킴 |
| 보관함 | 1 | — |

### 3-1. 보물 카드 35장

**일반 보물 28장**: 4색(노랑·빨강·청록·파랑) × 숫자 1~7. 같은 색은 모두 같은 그림. 각 (색, 숫자) 조합은 1장씩 존재.

**특수 카드 7장**:

| 카드 | 수량 | 효과 |
|------|------|------|
| 굉장한 보물 | 1장 | 와일드. 원하는 색·원하는 숫자의 보물 카드로 간주하여 세트에 사용할 수 있다. |
| 슬쩍하기 | 1장 | 획득하는 즉시(분배 단계) 이 카드와 버림 더미의 카드를 전부 카드 더미에 합쳐 섞은 뒤, 그 팀이 카드 더미에서 1장을 뽑아 가져온다. |
| 저주받은 그림 | 5장 | 각 카드에 숫자 2~6이 하나씩(2·3·4·5·6). 세트에 넣을 수 없다. 처분하려면 2코인을 지불해야 한다. 단, 카드 숫자와 같은 숫자의 보물 카드를 포함한 세트를 환금할 때는 코인 없이 이 카드 1장을 처분할 수 있다(숫자는 이 무료 처분 조건에만 사용). |

### 3-2. 간부 타일 (4종)

각 타일에는 ① 이름 ② 리더 효과 ③ 환금표(모든 간부 공통, §7) ④ 숫자 ⑤ 색깔이 적혀 있다. 리더 효과는 리더 토큰이 올라간 타일에서만 발동한다.

| 간부 | 리더 효과 |
|------|-----------|
| 플랭키 | 플랭키의 보물 보유 한도가 일반보다 1장 더 많다. |
| 척 | 팀이 한 라운드 동안 2번 이상 환금했다면 1코인을 얻는다. |
| 와이즈 | 환금 단계가 끝났을 때, 상대 팀이 저주받은 그림을 2장 이상 보유 중이면 1코인을 얻는다. |
| 죠 | 팀이 6코인 이상에 해당하는 세트를 환금했다면 1코인을 얻는다. |

### 3-3. 도우미 카드 (9종)

모두 환금 단계의 【C】 도움 요청으로 발동한다(뒷면 → 앞면). "조건부"는 해당 환금이 일어나는 순간에만 사용할 수 있다.

| 도우미 | 효과 | 분류 |
|--------|------|------|
| 음속의 쿠쿠 | 같은 색깔 카드 3장 이상을 포함한 세트를 환금했을 때 사용. 3코인을 얻는다. | 조건부 보너스 |
| 닌자 레오 | 같은 숫자 카드 3장 이상을 포함한 세트를 환금했을 때 사용. 3코인을 얻는다. | 조건부 보너스 |
| 명견 럭키 | 같은 색깔 카드 5장을 포함한 세트(= 연속+같은색 5장)를 환금했을 때 사용. 7코인을 얻는다. | 조건부 보너스 |
| 경찰견 알파 | 굉장한 보물 없이 숫자 1 보물 카드 4장을 포함한 세트를 환금하면, 그 팀은 게임에서 즉시 승리한다. | 조건부 즉승 |
| 샛길의 더그 | 팀이 보관 중인 보물 중 저주받은 그림이 아닌 카드를 원하는 만큼 처분한 뒤, 그만큼 카드 더미에서 뽑아 가져온다. | 행동 |
| 완력의 투스커 | 카드 더미에서 1장을 뽑아 가져온다. 다음 라운드가 시작될 때까지 팀의 보물 보유 한도가 사라진다. | 행동 |
| 척후 바이퍼 | 팀이 보관 중인 저주받은 그림을 코인 없이 모두 처분하고, 처분한 장수만큼 코인을 얻는다. | 행동 |
| 검은 날개 고물상 | 버림 더미에 있는 굉장한 보물을 가져온다. 단, 이번 라운드가 끝날 때까지 팀은 환금 행동을 할 수 없다. | 행동 |
| 크록 형제 | 앞면으로 놓인 도우미 카드 1장을 골라 그 카드의 효과를 사용한다(이미 사용된 효과 복사). | 행동 |

## 4. 게임 준비

1. **팀 나누기**: 두 팀으로 나눠 마주보고 앉는다. 각 팀은 동전 2코인씩 받고, 나머지 동전은 공용 더미로 둔다.
2. **간부 타일 배분**: "플랭키" 타일을 반드시 포함해 인원수만큼 준비하고, 각 플레이어에게 무작위로 1개씩 나눠 앞면으로 자기 앞에 놓는다.
3. **도우미 카드 선택**:
   - 도우미 9장을 뒷면으로 섞어 각 팀에 3장씩 나눠준다.
   - 각 팀은 받은 3장 중 2장을 선택해 팀원끼리 1장씩 나눠 가진다(상대에게 비공개, 뒷면으로 놓음). 1인 팀이면 그 플레이어가 2장 모두 가진다.
   - 선택하지 않은 카드와 남은 도우미 카드는 앞면을 보지 않은 채 상자에 반납한다.
4. **리더 결정**: 각 팀은 리더 1명을 정해 그 플레이어의 간부 타일 위에 리더 토큰을 놓는다(리더 효과 활성화). 리더가 아닌 플레이어는 멤버라 부른다.
5. **카드 더미**: 보물 카드 35장을 뒷면으로 섞어 양 팀의 손이 닿는 곳에 둔다.
6. **시작**: "플랭키" 타일 보유자가 속한 팀이 분할 팀, 상대가 선택 팀이 된다. 팀 표시 막대기의 화살표가 분할 팀을 향하게 놓고 1라운드를 시작한다.

**테이블 영역**: ① 팀 표시 막대기 ② 분배 영역(가운데) ③ 보물 보관 영역(플레이어마다 개별) ④ 카드 더미 ⑤ 버림 더미

## 5. 라운드 흐름

게임은 여러 라운드로 진행된다. 각 라운드는 4단계로 이루어지며, 라운드가 끝나면 분할 팀과 선택 팀의 역할을 교대한다.

```
[꾀부리기] → [분배] → [환금] → [종료] → (역할 교대) → 다음 라운드 …
```

> **카드 더미 소진 시**: 버림 더미를 뒷면으로 섞어 새 더미를 만든 뒤, 필요한 만큼 마저 뽑는다.

---

## 6. 단계별 상세

### 6-1. 꾀부리기 단계
- 막대기 화살표가 가리키는 팀이 분할 팀이 된다.
- 분할 팀 리더가 카드 더미에서 카드 5장을 뽑아 손에 든다(상대에게 비공개).
- 5장을 "2+3" 또는 "1+4"로 두 묶음으로 나눠 막대기 양옆에 놓는다.
- 5장 중 정확히 1장만 뒷면(비공개)으로 놓는다(2장 이상 불가). 어느 묶음에 둘지는 분할 팀이 자유롭게 정한다.
- 분할이 끝나면 막대기를 180° 회전시켜 선택 팀을 향하게 한다(주도권 이양).

### 6-2. 분배 단계
1. 선택 팀이 두 묶음 중 하나를 선택한다.
2. 선택한 묶음은 선택 팀이, 나머지는 분할 팀이 가져간다. 이때 뒷면 카드를 모두 앞면으로 공개한다.
3. 가져간 카드 중 슬쩍하기 카드가 있으면 즉시 그 효과를 처리한다.
4. 각 팀은 가져간 카드를 팀 내에서 자유롭게 나눠 가진다(한 명이 다 가져도 됨). 나눠 가진 카드는 각자의 보물 보관 영역에 놓는다.

### 6-3. 환금 단계 (모든 플레이어 동시 진행)
각자 아래 A·B·C를 원하는 순서로 원하는 횟수만큼 수행한다(아무것도 하지 않아도 된다). 모든 플레이어가 마치면 종료 단계로 넘어간다.

- **【A】 환금**: 자기 보관 영역의 카드로 세트를 만들어 버림 더미에 내고, 세트에 해당하는 코인을 얻는다.
  - 팀원의 카드와 합쳐서 세트를 만들 수는 없다(개인 보관 영역 안에서만).
  - 얻은 코인은 팀이 공유한다.
- **【B】 처분**: 자기 카드 1장을 버림 더미에 버린다. (저주받은 그림은 2코인을 내야 처분 가능 — §3-1)
- **【C】 도움 요청**: 뒷면으로 놓인 자기 도우미 카드를 앞면으로 뒤집어 효과를 사용한다. 이미 앞면으로 놓인 도우미 카드는 다시 사용할 수 없다(크록 형제 제외).

### 6-4. 종료 단계
1. **승리 확인**: 30코인 이상인 팀이 하나라도 있으면, 그 순간 코인이 더 많은 팀이 승리한다. (30코인 이상인 팀이 없거나, 양 팀 모두 30코인 이상이면서 동점이면 ②로 넘어간다.)
2. **보물 보유 한도 확인**: 보관 영역의 카드 수가 보유 한도를 초과하면, 한도 이하가 되도록 강제로 처분한다.

| 팀 구성원 수 | 보물 보유 한도 |
|--------------|----------------|
| 1명 | 5장 |
| 2명 | 리더 3장 / 멤버 4장 |

## 7. 세트 종류와 환금표

### 세트 종류

| 세트 | 조건 | 장수 | 비고 |
|------|------|------|------|
| 같은 숫자 | 숫자가 동일 | 2~4장 | 같은 숫자는 색이 모두 다름 |
| 연속된 숫자 | 숫자가 연속 (색 무관) | 3~5장 | 7 → 1 순환 불가 (예: 7,1,2 ✗) |
| 연속된 숫자 & 같은 색깔 | 숫자가 연속 + 색이 동일 | 3~5장 | 일반 연속보다 코인을 더 많이 얻음 |

### 환금표 (4개 간부 타일 공통)

| 세트 장수 | 같은 숫자 | 연속된 숫자 | 연속 + 같은 색 |
|:---------:|:---------:|:-----------:|:--------------:|
| 2장 | 2코인 | (불가) | (불가) |
| 3장 | 5코인 | 4코인 | 6코인 |
| 4장 | 10코인 | 6코인 | 9코인 |
| 5장 | (불가) | 10코인 | 15코인 |

- 같은 숫자: 2~4장만 가능 (2장=2, 3장=5, 4장=10).
- 연속된 숫자: 3~5장만 가능 (3장=4, 4장=6, 5장=10).
- 연속 + 같은 색: 3~5장만 가능 (3장=6, 4장=9, 5장=15).
